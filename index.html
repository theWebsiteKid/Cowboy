    Essential   4 Weeks

    React is a JavaScript library for building user interfaces. Developed @ Facebook, React popularized component architecture and a feature known as the Virtual DOM. React compares changes in state and only updates those components that were changed.

    Vue is a progressive JavaScript framework for building user interfaces. Developed by , an ex Googler who developed Angular 1.x, Vue combines the best of React, Angular 1.x, and Angular (Angular 2+). Like React, Vue can be dropped into an existing web project with a single script tag.

  • Angular

    Optional   2 Weeks

    Angular is a full feature, inclusive JavaScript framework developed and maintained by Google. Unlike React or Vue, which are libraries that can be paired with other tooling to create powerful apps, Angular takes the "batteries included" approach, and provides it's own libraries and tooling for almost everything. Angular is better suited for large, enterprise level apps. There is usually only one way to do any one thing, and this is referred to as "the Angular way".
